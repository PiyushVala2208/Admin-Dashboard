# 🛡️ AdminCore Dashboard v1.0

A premium, high-performance **Full Stack Admin Panel** designed for modern enterprise management. Built with a focus on speed and scalability.
---

## 📂 Project Structure

This repository is a **monorepo** containing both frontend and backend services:

* **`/client`** → Next.js 16 (App Router), Tailwind CSS v4, Lucide Icons
* **`/server`** → Node.js, Express API, Middleware Architecture

---

## 🚀 Quick Start Guide

Follow these steps to get the project running locally:

### 1️⃣ Clone Repository

```bash
git clone https://github.com/PiyushVala2208/Admin-Dashboard.git
cd Admin-Dashboard
```

---

### 2️⃣ Setup Backend (API)

```bash
cd server
npm install
npm run dev
```
---

### 3️⃣ Setup Frontend (UI)

```bash
cd ../client
npm install
npm run dev
```

---

## ✨ Key Features

* 🔐 **Role-Based Access Control (RBAC)**
  Manage users, roles, and permissions dynamically

* 📦 **Inventory Tracking**
  Real-time monitoring of stock levels and product data

* 📜 **Activity Logs**
  Timeline-based tracking of system actions

* 📱 **Ultra Responsive Design**
  Optimized for mobile, tablet, and large screens (4K support)

| Layer      | Technology                            |

## ⚙️ Environment Variables

Create a `.env` file inside `/server` and add:

```env
PORT=8000
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
```
